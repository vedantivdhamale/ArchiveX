# ArchiveX

ArchiveX is a console-based Java application that demonstrates the practical implementation of fundamental Data Structures through a file indexing and management system.

## Features

- Scan files from a folder
- Display all scanned files
- Search files by exact name
- Prefix-based file search using Trie
- Undo file deletion using Stack
- Backup queue using Queue
- Display folder hierarchy using an N-ary Tree

## Data Structures Used

- HashMap
- Doubly Linked List
- Stack
- Queue
- Trie
- N-ary Tree
- ArrayList

## How to Run

1. Clone the repository.
2. Open the project in IntelliJ IDEA.
3. Run `Main.java`.
4. Enter the folder path when prompted.
5. Use the menu to explore the features.
